This repository serves as an example for creating a C++ NuGet Package.

The NuGet package was created following the tutorial given at
http://coapp.org/tutorials/building-a-package.html

This C++ Repository houses a VS2012 solution that compiles with the vs110 tools.

In order to create the NuGet package for this repository navigate to the top level directory
using Windows PowerShell and input the command
Write-NuGetPackage .\jcstring.autopkg
